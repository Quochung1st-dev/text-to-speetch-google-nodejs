# Text-to-Speech Google Node.js

## Mô tả dự án
Dự án này là một ứng dụng web đơn giản sử dụng Node.js để chuyển đổi văn bản thành giọng nói. Người dùng có thể nhập văn bản vào giao diện web và nghe kết quả giọng nói được tạo ra.

## Tính năng
*   Chuyển đổi văn bản thành giọng nói.
*   Giao diện người dùng đơn giản và dễ sử dụng.

## Công nghệ sử dụng
*   **Backend:** Node.js
*   **Frontend:** HTML, JavaScript

## Hướng dẫn cài đặt

1.  **Clone repository:**
    ```bash
    git clone https://github.com/Quochung1st-dev/text-to-speetch-google-nodejs.git
    cd text-to-speetch-google-nodejs
    ```

2.  **Cài đặt các dependencies:**
    ```bash
    npm install
    ```

## Hướng dẫn sử dụng

1.  **Khởi động ứng dụng:**
    ```bash
    node index.js
    ```

2.  **Truy cập ứng dụng:**
    Mở trình duyệt của bạn và truy cập vào `http://localhost:3000` (hoặc cổng mà ứng dụng đang chạy).

3.  **Sử dụng giao diện:**
    Nhập văn bản vào ô nhập liệu và nhấn nút "Chuyển đổi" để nghe giọng nói được tạo ra.
